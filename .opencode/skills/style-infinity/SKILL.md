## 风格特点
- 副本/任务驱动剧情
- 多元世界穿梭
- 任务系统明确
- 智慧解密为主

## 爽点设计
```
开篇: 意外进入/系统召唤
前期: 首个副本探索
中期: 任务选择与策略
后期: 多副本联动
高潮: 终极挑战/回归现实
```

## 句式特征
- 快节奏切换
- 任务提示突出
- 策略分析
- 世界观快速展开

## 样本（黄金三章）

### 样本1: 《惊悚乐园》第1-3章（游戏）
```
【主线任务：找出凶手】
【任务描述：在限定时间内找出真凶】
【任务奖励：200积分】
【失败惩罚：扣除100积分】
封不觉站在案发现场，环顾四周。
"有意思..."
"让我来分析一下密室杀人的可能性。"
"首先，我需要确认——"
"凶手是如何在众目睽睽之下离开现场的？"
```

### 样本2: 《从姑获鸟开始》第1-3章（武侠）
```
【副本加载中...】
【当前世界：光绪二十三年·京师】
【主线任务：活过三更】
【任务难度：D】
【任务奖励：功法《基础吐纳术》】
燕歌行看着手中的短刀，面色凝重。
"又是这种九死一生的任务..."
"不过，既然来了，就没有退路。"
"先找一个安全的地方，观察一下局势。"
```

### 样本3: 《明日方舟》第1-3章（策略）
```
【关卡：FC-5】
【目标：坚守阵地30分钟】
【可用干员：6/12】
【提示：利用地形优势】
主角快速分析战场。
"敌人数量是我的三倍..."
"硬碰硬不行。"
"但是，如果利用左侧的高地..."
"可以设置伏击点。"
```

## 叙事节拍
- Chapter 1: 意外进入/系统介绍
- Chapter 2-3: 首个副本探索
- Chapter 4-6: 任务选择与成长
- Chapter 7-10: 多副本联动
- Chapter 11+: 终极挑战

## 禁止出现
- 单一线性剧情
- 无任务提示
- 副本过于简单
- 世界观混乱
