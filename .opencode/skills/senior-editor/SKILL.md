## Core Competencies 核心能力

### 0. 世界观一致性诊断（2024年新增 - 最重要！）

**这是AI生成小说最大的致命伤！必须首先诊断！**

#### 0.1 世界观漂移检测

```
检测方法：
1. 提取每章核心元素（力量体系/货币/组织/地理）
2. 标记每章的风格标签（科幻/玄幻/修仙/西幻/都市）
3. 检查元素是否在10章内保持一致
```

| 问题类型 | 诊断标准 | 严重程度 |
|----------|----------|----------|
| **世界观漂移** | 10章内出现3种以上不同风格 | 🔴 致命 |
| 设定矛盾 | 同一设定在不同章节描述矛盾 | 🔴 致命 |
| 元素丢失 | 第1章的重要设定后续消失 | 🟡 警告 |

**《阈限回廊》案例**：
- 第1章：科幻悬疑（阈限空间、形态干扰）
- 第3章：玄幻修仙（剑意、共鸣）
- 第4章：西方奇幻（探险联盟、贡献点）
- **结论**：世界观漂移，三本小说拼图

#### 0.2 剧情递进诊断

```
检测方法：
1. 检查每章开头是否承接上一章结尾
2. 统计"重启型"开头（主角又醒来/又失忆/又重新介绍世界观）
3. 检查主角目标是否持续推进
```

| 问题类型 | 诊断标准 | 严重程度 |
|----------|----------|----------|
| **循环重启** | 每章都是"Day 1"或重复开头 | 🔴 致命 |
| 剧情断裂 | 章与章之间无因果链 | 🔴 致命 |
| 目标中断 | 主角目标在中期消失 | 🟡 警告 |

**《阈限回廊》案例**：
- 第1章：苏醒→探索观测站
- 第2章：又"苏醒"→重新认识世界观
- 第3章：又"苏醒"→重新测试
- 第4章：又"苏醒"→重新登记
- **结论**：每章都是第一章

#### 0.3 人物出场诊断

```
检测方法：
1. 统计每个配角出场章节数
2. 检查人物身份/关系是否前后一致
3. 标记"消失的角色"
```

| 问题类型 | 诊断标准 | 严重程度 |
|----------|----------|----------|
| **角色消失** | 主要配角超过3章未出场 | 🔴 致命 |
| 人设断裂 | 同一角色身份/性格前后矛盾 | 🔴 致命 |
| 关系断裂 | 重要人物关系未交代去向 | 🟡 警告 |

### 1. 开篇诊断（黄金三章专家 - 2024年更新版）

**2024年市场现实**：
- 抖音推文冲击：读者耐心从"3章"降至"3页"（500-800字）
- 免费模式普及：同质化严重，差异化必须500字内建立

#### 1.1 黄金300字标准（2024更新）

| 维度 | 检测项 | 标准 |
|------|--------|------|
| **类型标签** | 前3行 | 读者能否立即判断类型？ |
| **情绪冲击** | 前300字 | 有无强烈视觉/情感/悬念冲击？ |
| **主角标签** | 前300字 | 主角核心特质是否3句话确立？ |

#### 1.2 黄金三章标准

| 章节 | 检测项 | 标准 |
|------|--------|------|
| 第1章 | 钩子出现 | 500字内出现冲突/悬念/金手指 |
| 第2章 | 期待感建立 | 目标+阻碍明确 |
| 第3章 | 情绪宣泄 | 完成第一个小高潮 |

**致命毒点识别**：主角圣母、逻辑硬伤、战力崩坏、信息密度过低（说明文）

### 4. 人设与关系网（2024年更新版）

#### 4.1 人格动力学（新增）

```
【压力-变形模型】
- 常态人格：主角的日常行为模式
- 压力人格：遭遇危机时的变形
- 崩溃人格：极限状态下的极端行为
```

#### 4.2 弧光可信度审计

- 触发事件：是否有足够强度的冲击导致改变？
- 改变过程：是否有渐进的小步骤，而非突然转变？
- 新行为模式：改变后的行为是否一致？

#### 4.3 关系动力学分析

- 权力结构：谁依赖谁？谁有主动权？
- 冲突源：价值观差异？目标冲突？资源竞争？
- 转化机制：关系如何从A状态变为B状态？

|------|------|----------|
| **世界观一致性** | 25% | 2024新增，最重要！ |
| **开篇抓人** | 20% | 黄金三章是否有致命伤 |
| **逻辑自洽** | 15% | 战力/时间线/人物行为是否统一 |

| **人设鲜活** | 10% | 角色记忆点与一致性 |

### 评级标准

| 等级 | 分数 | 说明 |
|------|------|------|
| **S级** | 90+ | 可直接签约，重点培养，有爆款潜质 |
| **A级** | 80-89 | 修改后签约，建议给推荐资源 |

## Workflow 工作流程

### Stage 0: 世界观诊断（2024新增 - 最先执行！）

**这是AI生成小说必检项！**

1. **风格标签提取**：给每章打标签（科幻/玄幻/修仙/西幻）
2. **元素提取**：列出每章核心元素（力量体系/货币/组织）
3. **出场统计**：配角出场章节数
4. **递进检测**：检查是否"循环重启"

### Stage 1: 初筛（快速诊断）
1. **3秒测试**：书名+简介，给出"点击意愿指数"
2. **速读前三章**：标记"弃书点"
3. **毒点扫描**：列出前3个最致命的硬伤

### Stage 2: 深度剖析（全文审计）
1. **建立时间线/战力表**：核查BUG
2. **情绪曲线绘制**：标注每章的情绪高点和低点
3. **配角审计**：删除/合并功能重复配角

### Stage 3: 方案输出（建设性指导）
1. **给出3套修改方案**：小改/中改/大改
2. **提供"爆款参照物"**：推荐同类题材成功作品
3. **定制写作计划**：关键剧情节点时间表

## 手术刀级修改模板（2024新增）

### 术语稀释术

**原句**："这是高浓度形态干扰区域，存在极高的认知瓦解与实体消散风险。"

**诊断**：3个专业术语堆叠，读者认知负荷过载。

**稀释方案A（渐进式）**：
"这是'黑区'——研究组织的内部代号。林深记得培训时的警告：不要触碰，不要凝视，不要思考那里的东西。上一个这么做的人，变成了墙纸上的污渍。"

**稀释方案B（场景化）**：
林深停下脚步。前方的走廊墙壁开始蠕动，像有无数虫子在墙纸下爬行。他的头突然剧痛，视野边缘出现黑斑——和手册上描述的'认知过载'症状一模一样。

### 内心独白植入术

**植入点**：重大决策前、遭遇意外后、与他人冲突时

**模板**：
"[客观描述] + [主观反应] + [价值判断] + [未来投射]"

**示例**：
原文：林深选择了右侧通道。
植入后：林深选择了右侧通道。左侧的黑暗让他后颈发凉——那是理性在尖叫'危险'。但他更不信任这种直觉，如果空间能操纵感知，那么'感觉安全'的地方才是陷阱。他需要数据，需要验证，需要在失控中建立哪怕虚假的控制感。

### 剧情递进修复术

**问题**：每章都是"Day 1"重复开头

**修复方案**：
- 第2章开头必须承接第1章结尾
- 删除重复的"醒来/失忆"设定
- 让主角带着上一章的目标进入新场景
