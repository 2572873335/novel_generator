## 风格特点
- 主角谨慎稳健，不轻易冒险
- 延迟满足，厚积薄发
- 心理活动丰富，风险计算
- 表面低调，暗中发育

## 爽点设计
```
开篇: 稳健入门/不冒头
前期: 暗中观察，积累资源
中期: 把握机会，一鸣惊人
后期: 实力曝光，震惊全场
高潮: 以弱胜强，稳健反击
```

## 句式特征
- 长句较多，思考分析
- 心理活动占比高
- 动作描写相对较少
- 强调风险评估和利弊分析

## 样本（黄金三章）

### 样本1: 《苟道》第1-3章（仙侠）
```
"此事还需从长计议。"
方源眼神深邃，仔细权衡着每一种可能的风险。
在做出决定之前，他需要确认：
一、对手的实力是否在可控范围
二、失败的后果能否承受
三、最坏情况下能否全身而退
...
"稳妥起见，再观察三日。"
```

### 样本2: 《我只想安静地玩游戏》第1-3章（游戏）
```
【检测到精英怪物：暗影刺客】
【战力评估：玩家当前实力 72 vs 怪物战力 85】
【建议：规避/呼叫支援】
林墨却没有立即离开。
"等等，让我分析一下..."
"它的攻击模式是三连击，有0.5秒的后摇。"
"如果能抓住这个间隙..."
```

### 样本3: 《黎明之剑》第1-3章（科幻）
```
精灵卫星项目的可行性分析报告（节选）
- 技术成熟度：LEVEL 3（实验室验证）
- 预计研发周期：2.4年
- 所需资源：详见附录B
- 风险评估：中等可控
"在没有足够工业基础的情况下，"
"我们必须用更长的时间来换取更大的成功率。"
主角这样写道。
```

## 叙事节拍
- Chapter 1: 观察环境，确认规则
- Chapter 2-3: 暗中布局，积累资源
- Chapter 4-6: 小试牛刀，验证实力
- Chapter 7-10: 把握机会，一鸣惊人
- Chapter 11+: 实力曝光，稳健反击

## 禁止出现
- 冲动冒险、不做保险
- 无脑冲锋、不计后果
- 意气用事、情绪用事
- 明明可以稳赢非要浪
