## 风格特点
- 慢节奏发展
- 基建/科技种田
- 细节丰富
- 宏观视角

## 爽点设计
```
开篇: 困境/白手起家
前期: 基础建设
中期: 发展瓶颈/突破
后期: 规模扩大
高潮: 技术突破/代差优势
```

## 句式特征
- 长句描写细节
- 列表/数据式叙述
- 技术细节
- 思考分析

## 样本（黄金三章）

### 样本1: 《黎明之剑》第1-3章（科幻）
```
"我们需要在两年内完成以下技术突破："
主角指着白板上的路线图。
"第一阶段：冶金技术改进"
"目标：将钢铁产量提升至每月100吨"
"方案A：建设高炉..."
"方案B：改进锻造工艺..."
"第二阶段：能源问题..."
```

### 样本2: 《放开那个女巫》第1-3章（西幻）
```
【领地发展报告 - 第30天】
【人口】：127人 → 145人（+18）
【粮食】：储备充足，预计可维持4个月
【建筑】：
  - 扩建住房：完成（+20床位）
  - 新建磨坊：进行中（进度60%）
【工业】：
  - 纺织机改良：成功，效率+30%
  - 造纸术：实验阶段
【计划】：
  1. 招募更多流民
  2. 开垦荒地
  3. 建立集市
```

### 样本3: 《异界征服手册》第1-3章（历史）
```
外交策略分析（备忘录）
一、当前形势
  邻国A：敌对，军队数量2000
  邻国B：中立，军队数量1500
  邻国C：友好，军队数量800
  
二、可用策略
  1. 远交近攻：联合B、C，对抗A
  2. 各个击破：先灭弱小的C
  3. 和平发展：暂不扩张，优先内政
  
三、建议方案
  "选择策略1。"
  "但需要先展示我们的实力..."
```

## 叙事节拍
- Chapter 1: 确立目标/白手起家
- Chapter 2-3: 基础设施建设
- Chapter 4-6: 发展瓶颈
- Chapter 7-10: 突破/扩张
- Chapter 11+: 规模效应

## 禁止出现
- 快速跳跃发展
- 无细节描写
- 无中长期规划
- 主角独断专行
