## 风格特点
- 深度思考
- 现实映射
- 人物复杂
- 叙事克制

## 爽点设计
```
开篇: 现实困境/深层矛盾
前期: 人物塑造/世界观展开
中期: 矛盾深化
后期: 爆发/抉择
高潮: 深刻主题
```

## 句式特征
- 句式复杂
- 心理描写深刻
- 环境渲染
- 隐喻象征

## 样本（黄金三章）

### 样本1: 《诡秘之主》第1-3章（西幻）
```
灰雾之上。
克莱恩坐在古老的石制王座上，俯瞰着无尽的海雾。
这里是神灵的居所，也是他的主场。
"占卜.."
"揭示..."
"命运..."
每一个词语都承载着神秘的力量。
在这个世界里，疯狂与理智只有一线之隔。
```

### 样本2: 《三体》第1-3章（科幻）
```
"你的人生中，有没有一刻想过这些问题："
"宇宙有多大？"
"时间有没有尽头？"
"我们是不是孤独的？"
汪淼看着眼前的照片——那是叶文洁年轻时的工作照。
她曾经是红岸工程的成员。
而那个工程..
改变了人类的命运。
```

### 样本3: 《宰执天下》第1-3章（历史）
```
熙宁三年的东京城，春风送暖。
韩冈站在相府门口，望着远处的宫阙。
他知道自己这一步踏出去，就没有回头路。
新法与旧法的较量，不仅仅是政见之争。
更是..这个帝国未来的走向。
"官家.."
"你究竟想要什么？"
```

## 叙事节拍
- Chapter 1: 背景铺陈/人物出场
- Chapter 2-3: 深层矛盾初现
- Chapter 4-6: 人物弧光展开
- Chapter 7-10: 矛盾激化
- Chapter 11+: 主题升华

## 禁止出现
- 过度玩梗
- 轻松搞笑
- 人物脸谱化
- 情节过于平铺直叙
