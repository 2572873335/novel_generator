## 风格特点
- 主角道德灰色地带
- 利益至上，理性决策
- 世界观残酷，资源匮乏
- 杀伐果断，不圣母

## 爽点设计
```
开篇: 残酷觉醒/利益交换
前期: 不择手段获取资源
中期: 利用一切可用之力
后期: 背叛与被背叛
高潮: 利益最大化or情感抉择
```

## 句式特征
- 冷静客观的叙述
- 利益计算式心理活动
- 残酷场景不过度渲染
- 道德灰色地带讨论

## 样本（黄金三章）

### 样本1: 《蛊真人》第1-3章（仙侠）
```
"人活着，就是为了更好地活着。"
方源的利益公式很简单：
收益 - 成本 - 风险 = 净利润
"亲情？友情？爱情？"
"不过是獲取利益的工具罢了。"
"若有一天，它们成为阻碍..."
"舍弃便是。"
```

### 样本2: 《轮回乐园》第1-3章（无限）
```
【任务目标：击杀暗精灵公主】
【奖励：500乐园币】
【失败惩罚：抹杀】
苏晓点头。
"这个任务，我接了。"
"不过，我需要先确认一件事——"
"暗精灵公主的弱点是什么？"
"以及，杀死她之后，我能得到多少额外收益？"
```

### 样本3: 《熔流》第1-3章（科幻末世）
```
资源分配方案（最终版）
A级资源：武器、燃料 → 战斗人员
B级资源：食物、水 → 所有人定量
C级资源：药品 → 伤员优先
"这是最理性的分配方式。"
"如果有人反对..."
"那就让他离开基地。"
"毕竟，末世不需要弱者。"
```

## 叙事节拍
- Chapter 1: 残酷觉醒/规则确认
- Chapter 2-3: 第一次利益交换
- Chapter 4-6: 道德底线测试
- Chapter 7-10: 背叛与成长
- Chapter 11+: 利益最大化or情感回归

## 禁止出现
- 圣母心、伟光正
- 无意义善良
- 主角突然正义感爆棚
- 敌人突然良心发现
