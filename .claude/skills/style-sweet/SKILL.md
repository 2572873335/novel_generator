## 风格特点
- 轻松甜蜜
- 情感互动
- 互相治愈
- 温馨日常

## 爽点设计
```
开篇: 相遇/心动
前期: 甜蜜互动
中期: 误会/表白
后期: 确认关系/撒糖
高潮: 甜蜜时刻/承诺
```

## 句式特征
- 短句+对话
- 心理描写
- 细节温暖
- 互动感强

## 样本（黄金三章）

### 样本1: 《我家老婆来自一千年前》第1-3章（都市）
```
"那个...你叫什么名字？"
姜禾看着眼前的女孩，有些紧张。
"我叫..."
女孩歪着头想了想。
"我叫姜禾呀。"
"你..你好。"
"嗯，你好。"
女孩笑了，眼睛弯成月牙。
这一刻，阳光正好。
```

### 样本2: 《光阴之外》日常篇第1-3章（玄幻校园）
```
"这道题 ..."
许青敲了敲脑袋，还是没有头绪。
"这里要用换元法。"
旁边递过来一张草稿纸。
"谢谢..."
他抬头，看见了一张明媚的笑脸。
"不客气呀。"
"我叫许青。你呢？"
```

### 样本3: 《学渣的心理咨询师》第1-3章（都市）
```
"你的问题，我理解了。"
心理医生温柔地看着眼前的少年。
"其实.."
"你不需要逼迫自己。"
"每个人都有自己的节奏。"
"慢慢来。"
少年愣住了。
"可是..我怕让父母失望..."
"那就为他们，更是为自己。"
"你已经很努力了。"
```

## 叙事节拍
- Chapter 1: 相遇/初见
- Chapter 2-3: 接触/了解
- Chapter 4-6: 心动的瞬间
- Chapter 7-10: 甜蜜互动/小冲突
- Chapter 11+: 确定关系/撒糖

## 禁止出现
- 强行虐恋
- 刀多于糖
- 过度误会
- 狗血剧情
